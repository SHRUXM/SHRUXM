## About

I am a Computer Science student focused on Cyber Security and AI-driven defense systems.  
My primary interest lies in **Intrusion Detection Systems, Network Traffic Analysis, and Explainable AI for security applications.**

I aim to build secure, interpretable, and real-world applicable security solutions.

## Featured Project

### Explainable Intrusion Detection System (GCN)

- Dataset: **UNSW-NB15**
- Architecture: **Graph Convolutional Network**
- Framework: **PyTorch & PyTorch Geometric**
- Explainability: **Captum (Integrated Gradients)**

Key Work:
- Network traffic preprocessing  
- Graph construction using protocol similarity  
- GNN-based attack classification  
- Feature attribution for attack interpretation
